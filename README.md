A premium, interactive web application designed to help students and IT professionals study for the **CompTIA Network+ (N10-009)** certification. Built with modern web technologies, it features a "Deep Space" dark mode, interactive diagrams, and a comprehensive curriculum covering all exam objectives.

## üöÄ Features

*   **Comprehensive Curriculum**: Covers all 5 domains of the N10-009 exam (Networking Fundamentals, Implementations, Operations, Security, Troubleshooting).
*   **Interactive Quizzes**: "Teach-then-Test" methodology with immediate feedback and detailed explanations for every topic.
*   **Visual Learning**: Animated diagrams for complex concepts like OSI Model, Encapsulation, and TCP Handshakes.
*   **Premium UI**: Glassmorphic design with a custom "Deep Space" theme to reduce eye strain during long study sessions.
*   **Responsive**: Works seamlessly on desktops, tablets, and mobile devices.

## üì¶ Installation & Setup

Follow these steps to get the project running locally on your machine.

### Prerequisites

*   **Node.js**: Version 16.0 or higher. [Download Here](https://nodejs.org/)
*   **Git**: [Download Here](https://git-scm.com/)

### 1. Clone the Repository

```bash
git clone https://github.com/minatolabs/learn-network.git
cd learn-network
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the Development Server

**Windows:**
```powershell
npm run dev
```

**Linux / macOS:**
```bash
npm run dev
```

The application will start at `http://localhost:5173`.

### 4. Build for Production

To create an optimized build for deployment:

```bash
npm run build
```

The artifacts will be generated in the `dist/` directory.
